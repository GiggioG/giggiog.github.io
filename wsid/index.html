What is WSID

WSID is a system for knowing in what writing system I have quickly written something.
I am a person that frequently looks around in r/neography and similar places and likes
to try the scripts that people create, but I also like knowing what I wrote in some
obscure script 2 months ago. That's why I created WSID. Every time before I try a new script,
I add a new entry to the google sheet with the name of the script, its ID and a link to
where it's documented. I usually try these scripts on paper, so somewhere around the text
I wrote with the script, I draw the following symbol, followed by the WSID of the script I'm
trying. Like in this example, sitelen sitelen (WSID 8):

The symbol has two parts, with the

the bottom of the WS part is optional. The second part is a ligature of a lowercase i
and a lowercase d. For now only I can add writing systems to WSID, because I made it for
myself, but I encourage everyone to create a similar system for themselves. If you do,
then PLEASE don't use my symbol, because that will create lots of confusion. If enough
people create their own systems for documenting scripts, then we might need WSIDID - a system
for documenting WSID systems, but that's far off into the future for now...
